Are you looking to learn Computer programming? Are you looking to expand on your current coding skills? Well this is the right place for you because I am going to take through the basics of programming in Microsoft C#.

Microsoft C# is a multipurpose computer programming language suitable for all development needs. using C# you can develop software application for Microsoft Windows family of operating systems as well as other non-Microsoft products such as Unix, Ubuntu, Linux and Mac OS as long as they have .NET installed on them.

What is .NET? and What is its purpose or role in computer programming?

.NET is a software development platform for Windows, Mac OS, Linux and other non-Microsoft operating systems. Programming languages such as VB, C# and Microsoft C++ all target or make use of this platform to run their applications on our computers.

.NET is composed of three components (CLS, CTS and CLR).

Microsoft Visual Studio is a tool that integrates all these into a single place to make the programming journey licking good. On this course we go through the details to get you started into your coding/programming journey. We explain the components of .NET (CLS, CTS and CLR).
